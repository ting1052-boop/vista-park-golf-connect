# 직접생산 확인 증빙 자료

프로그램명: VISTA Park Golf Connect · 작성일: 2026-07-02

## 1. 직접생산 사실 요약

본 프로그램은 (기입: 업체명 — HH Square)이 기획·설계·개발·시험·배포의
전 과정을 직접 수행한 자체 개발 소프트웨어이다.

- 외주 개발 없음. 타사 상용 솔루션의 재판매·개조가 아님.
- 소스코드 전체를 자체 보유하며 버전관리 시스템(Git)으로 개발 전
  과정이 기록되어 있음.
- 프로그램 명칭, 화면 구성, 업무 로직, 데이터베이스 설계 전부 자체
  창작물임.

## 4. 사용 오픈소스 및 라이선스

프로그램은 아래 공개 라이선스 라이브러리를 도구로 사용한다.
모두 상용 이용이 허용된 라이선스이며, 프로그램의 창작적 표현
(화면, 업무 로직, DB 설계)은 이에 포함되지 않는 자체 저작물이다.

## 5. 자체 시험 수행

개발 과정에서 수행한 기능 시험의 시나리오와 실제 결과는
`docs/test-scenarios.md`에 기록하였다. 대표 항목:

- 고객 예약 전체 흐름 (입력 → 확인 → 확정 → DB 저장) 실기 검증
- 이중예약 차단: 응용 계층·DB 계층 각각 실제 재현 시험으로 확인
- 관리자 인증: 미인증 접근 시 로그인 화면 리다이렉트 확인
- PWA: 매니페스트/아이콘 응답, 홈 화면 설치 확인
