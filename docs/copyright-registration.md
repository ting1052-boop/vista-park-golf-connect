## 2. 프로그램의 목적 및 기능 요약

VISTA Park Golf Connect는 스크린파크골프 가맹점의 운영을 지원하는 자체 개발
웹 프로그램이다. 다음 기능을 하나의 시스템으로 통합 제공한다.

1. **고객 모바일 예약** — 매장 선택, 날짜·시간 선택, 인원·이용시간 선택,
   타석 자동 배정, 예약 확인 요약, 예약 확정. PWA 방식으로 휴대전화 홈
   화면에 설치하여 앱처럼 사용 가능.
2. **예약 충돌 방지** — 응용 계층의 실시간 가용 타석 계산과 데이터베이스
   계층의 배타 제약조건(exclusion constraint)을 이중으로 적용하여 동일
   타석·동일 시간대 이중예약을 원천 차단.
3. **매장 관리** — 매장, 타석, 장비, 회원 정보의 등록·조회·수정·삭제(CRUD)
   를 Supabase PostgreSQL과 실시간 연동.
4. **관리자 대시보드** — 타석 이용 현황, 예약 현황, 운영 알림을 단일
   화면에서 관제.
5. **무인 운영 준비 구조** — 예약 시각 5분 전 매장 장비(조명, 프로젝터,
   타석 PC 전원)를 자동 준비하기 위한 API 구조 및 Home Assistant 연동
   브리지.

## 3. 창작성 및 독립성

- 본 프로그램의 전체 소스코드는 저작자가 직접 작성하였으며, 타사 상용
  프로그램의 소스코드를 복제하거나 개조한 부분이 없다.
- 화면 구성(UI), 브랜드 명칭(VISTA Park Golf Connect), 아이콘, 색상 체계
  (밝은 배경 + VISTA 그린 #4E8969)는 자체 설계이다.
- 사용한 외부 구성요소는 모두 공개 라이선스(MIT, Apache-2.0)의 오픈소스
  라이브러리이며, 목록과 라이선스는 `direct-production-evidence.md` 4절에
  명시하였다. 오픈소스 라이브러리는 도구로 사용되었을 뿐 프로그램의
  창작적 표현(화면 구성, 업무 로직, 데이터 구조)은 전부 자체 저작물이다.
- 개발 전 과정이 Git 버전관리 이력(2026-06-14 최초 커밋부터)으로 보존되어
  있어 창작 과정을 시점별로 증빙할 수 있다.
