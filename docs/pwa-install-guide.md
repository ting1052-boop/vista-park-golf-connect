# VISTA Park Golf Connect PWA 설치 안내

## 목적

VISTA Park Golf Connect는 1차 MVP에서 정식 앱스토어 앱 대신 PWA 방식으로 제공합니다.
고객은 모바일 브라우저에서 접속한 뒤 홈 화면에 추가해 앱처럼 사용할 수 있습니다.

## 고객 설치 방법

### Android Chrome

1. VISTA 예약 주소에 접속합니다.
2. 화면에 표시되는 `앱 설치` 버튼을 누릅니다.
3. 브라우저 안내에 따라 설치를 완료합니다.
4. 휴대폰 홈 화면의 `VISTA Connect` 아이콘으로 실행합니다.

### iPhone Safari

1. Safari에서 VISTA 예약 주소에 접속합니다.
2. 하단 공유 버튼을 누릅니다.
3. `홈 화면에 추가`를 선택합니다.
4. 이름을 확인한 뒤 추가합니다.

## 1차 MVP 범위

- 고객 예약 화면 홈 화면 설치
- 관리자 PC 웹 화면 유지
- HTTPS 배포 주소 기반 실행
- 예약/매장/타석 데이터는 Supabase와 연결

## 향후 정식 앱 전환 기준

- 가맹점 수 증가
- 예약 고객 사용량 증가
- 푸시 알림 필요
- 위치 기반 가까운 매장 검색 필요
- 결제 연동 필요

정식 앱 전환 전까지는 PWA 방식으로 예약 흐름과 매장 운영 흐름을 먼저 검증합니다.
